# timelagged-mb-interactions

This repository contains an unfinished research workflow developed to reassess microbial interaction networks in densely sampled longitudinal 16S rRNA microbiome data. 
The workflow was created as a follow-up to a [study previously posted to bioRxiv](https://www.biorxiv.org/content/10.1101/2025.02.27.639550v1) in which significance of pairwise interactions was evaluated using random permutation-based null models. The primary goal of this project was to replace that approach with Iterative Amplitude Adjusted Fourier Transform (IAAFT) surrogate time series to preserve both the amplitude distribution and autocorrelation structure of the original data in the null models. 
Development of the downstream analyses has not been completed, and therefore, this repo is provided as a documented research resource rather than finished software package or companion to publication. 
The code, intermediate outputs, and analyses are organized here to support future development by other researchers.
